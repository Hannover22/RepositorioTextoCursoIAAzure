# RepositorioTextoCursoIAAzure

O Language Studio do Microsoft Azure é uma plataforma web sem código para criar, testar e implantar soluções de inteligência artificial para linguagem natural.
Principais funções:

Testar recursos prontos: análise de sentimento, extração de entidades, frases-chave, detecção de idioma e tradução em tempo real.

Criar modelos personalizados: classificação de texto e extração de entidades específicas do seu negócio, apenas enviando exemplos rotulados.

Question Answering: transformar documentos ou sites em bases que respondem perguntas em linguagem natural.
Conversational Language Understanding (CLU): entender intenções e entidades para chatbots.

Orchestration Workflow: combinar várias tarefas em um único fluxo.

Resumização e geração de texto (integrado ao Azure OpenAI).

Você conecta um recurso Azure AI Language, faz tudo pela interface visual, treina, avalia e implanta o modelo com um clique, obtendo uma API REST pronta para usar. É rápido, não exige programação e atende padrões empresariais de segurança.
